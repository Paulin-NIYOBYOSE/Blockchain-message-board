# simple message board

## Getting started 

### `Blockchain part`

```sh
#first clone the repo and go in the root folder 
# installation and setup
npm install -g truffle@latest
# initialize project
truffle init
# build project
truffle compile
# Test
truffle test
# deploy
truffle deploy
```

### `Client part`

```sh
#Enter the client folder to run the frontend 
cd client
# install dependencies
npm install
# Run the app
npm run dev
```
